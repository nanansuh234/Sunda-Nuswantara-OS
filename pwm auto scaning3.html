SKEMA TRANSMISI GAS HIDROGEN (H2)

Jalur Aman dari Reaktor Resonansi ke Titik Bakar

🚰 Perangkat Jalur

  • Selang: Polyurethane (PU) atau selang bening tebal anti-kempis.
  • One-Way Valve: Katup searah agar gas tidak balik.
  • Flashback Arrestor: Wajib ada untuk mematikan api balik.

🔥 Titik Guna

  • Kompor: Modifikasi spuyer gas menjadi lebih kecil.
  • Intake Mesin: Masukkan selang sebelum filter udara.

🛡️ Protokol Keamanan (Flashback Guard)

1. PRIMARY BUBBLER: Gas dari reaktor masuk ke botol berisi air. Gas keluar lewat gelembung. Ini sekat pertama jika api menyambar balik.
2. SECONDARY BUBBLER: Gunakan botol kedua sebagai cadangan keamanan ekstra (Double Guard).
3. DRY FILTER: Gunakan tabung berisi sabut baja halus (Steel Wool) sebagai penahan api terakhir sebelum ke selang utama.

🚀 Alur Instalasi (Foot Step)

Hubungkan reaktor ke Bubbler 1 ➔ Bubbler 2 ➔ One-Way Valve ➔ Titik Bakar.
Nyalakan alat resonansi, tunggu gelembung stabil, baru nyalakan api di ujung nozzle/kompor.
