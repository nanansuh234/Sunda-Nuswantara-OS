HIGH-POWER PWM MODULE (3D PRINT INTEGRATION)

Driver Resonansi Skala Besar untuk Manipulasi Logam

⚡ Komponen Daya (The Muscle)

  • IGBT Module: Tipe FGA25N120 (Tahan 1200V / 25A) atau MOSFET IRFP460.
  • Gate Driver: IC TLP250 atau IR2110 (Wajib pakai Optocoupler agar Arduino tidak meledak).
  • Heatsink + Fan: Gunakan pendingin aktif (kipas server) karena switching frekuensi tinggi bikin panas luar biasa.

🤖 Integrasi 3D Printer

  • Control: Hubungkan Pin Heater Out atau Fan PWM dari Board 3D Printer (RAMPS/SKR) ke input Driver ini.
  • Signal: Gunakan isolator optik untuk memisahkan tegangan 12V/24V printer dengan tegangan tinggi reaktor.

🔌 Alur Wiring (High-End Logic)

1. Logic Input: Signal PWM dari Board Printer ➔ Optocoupler (TLP250).
2. Gate Drive: Optocoupler ➔ Menggerakkan Gate IGBT dengan tegangan 15V stabil.
3. Power Loop: Sumber daya DC (Aki/PSU Besar) ➔ Melalui IGBT ➔ Menuju Transduser/Koil Resonansi di Nozzle Printer.
4. Feedback: Sensor Arus ACS712 lapor balik ke Arduino untuk Auto-Tuning frekuensi saat logam mulai "lumer".
