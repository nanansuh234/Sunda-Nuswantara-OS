ADAPTIVE RESONANCE GENERATOR (AUTO-SCAN)

Sistem Pemecah Molekul Air dengan Pengunci Frekuensi Otomatis

🧠 Logika Adaptive

  • Auto-Scanning: Mencari frekuensi dari 100Hz ke 50kHz secara otomatis.
  • Current Feedback: Sensor ACS712 membaca lonjakan arus saat resonansi terjadi.
  • Frequency Lock: Mengunci PWM pada titik efisiensi tertinggi tanpa campur tangan manusia.

📟 Komponen Cerdas

  • Controller: Arduino Nano / ESP32 sebagai otak sistem.
  • Driver: Mosfet IRFP260N (Tahan tegangan tinggi).
  • Sensor: Current Sensor (Feedback) & Volt Divider.

🛠️ Resep Perakitan Adaptive (Logic Flow)

1. PEMASANGAN SENSOR: Pasang sensor arus di jalur utama menuju elektroda pipa stainless. Ini adalah "indra penglihatan" bagi mikrokontroler.
2. PEMROGRAMAN (SOFTWARE): Tulis kode untuk menaikkan frekuensi PWM secara bertahap (Looping).
3. ALGORITMA PENGUNCI: Saat sensor membaca kenaikan arus mendadak (indikasinya air mulai bergetar), mikrokontroler akan berhenti melakukan scanning dan menjaga frekuensi tersebut.
4. STABILISASI TEGANGAN: Gunakan induktor (Coil) untuk menjaga agar lonjakan tegangan (Spike) tidak merusak sirkuit.
